# Brown Bags Tracker

A tool used for Brown Bag sessions to keep track of upcoming brown bag presentations. It centralizes information about each session, including:

- Title
- Date of presentation
- Presenter
- Basic background of the presenter
- Topic category
- Whether the presenter has done previous presentations

## Future Features

- Schedule a presentation directly from the app
- View information about past presentations
- View resources of past presentations (decks, repos, recordings) linked to the Confluence page where these resources are currently kept
